100 and 50 are operands and the symbols are the Operators. 

var sum1 = 100 + 50       // 150 (100 + 50)
var sum2 = sum1 + 250     // 400 (150 + 250)
var sum3 = sum2 + sum2    // 800 (400 + 400)

Assignment operators


var x = 10
x += 5
The addition assignment operator (+=) adds a value to a variable 

fun main() {
    var numero = 10

    numero += 5

    println(numero) // Resultado: 15
}

Comparison Operators


fun main() {
    val a = 10
    val b = 20

    println(a == b)  // false  (equal)
    println(a != b)  // true   (not equal)
    println(a > b)   // false  (greater than)
    println(a < b)   // true   (less than)
    println(a >= b)  // false  (greater or equal)
    println(a <= b)  // true   (less or equal)
}
Comparison operators are used to compare two values, and returns a Boolean value: either true or false. 

fun main() {
    val age = 18

    if (age >= 18) {
        println("You are an adult")
    } else {
        println("You are a minor")
    }
}

Logical Operators


fun main() {
    val a = true
    val b = false

    println(a && b) // false (AND)
    println(a || b) // true  (OR)
    println(!a)     // false (NOT)
}
Logical operators are used to determine the logic between variables or values 

fun main() {
    val age = 20
    val hasID = true

    if (age >= 18 && hasID) {
        println("You can enter")
    } else {
        println("Access denied")
    }
}
